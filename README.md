# CStage
CStage is a fast ASCII-based video player written in C++. It renders video frames as text directly inside the terminal, creating a retro console-style viewing experience. CStage is a rewrite of the original [PyStage](https://github.com/Tinnth7/CStage/tree/pystage) project, which was written in Python.

## CStage – ASCII Video Player

Turn any video into beautiful ASCII art – right in your terminal (with a GUI)!

---

## Requirements

### With the source code:
* C++17 or newer
* UTF-8 compatible terminal

### With the Windows Executable (.exe) file:
* At least Windows 10 1903 or newer
